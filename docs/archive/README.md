# Lightning Classics Archived Docs

This directory holds superseded but still useful working-history material.

Files belong here when:

- their durable guidance has already been absorbed into the canonical docs in `docs/`
- they are no longer the current operating source of truth
- keeping the history is still valuable for traceability, context, or auditability

Archive rule:

- move files here instead of deleting them unless there is a specific reason not to retain them
